Activism website template

Inspired by the BLM movement, I wanted a way for people with little or no coding experience to create a website for people who want to become educated and involved. With this React website template, any resources the creator wants to include (such as books, petitions, articles, and more) are easily organized into a user-friendly website. Each resource's information is simply inputed into the data files, and the code does the rest of the work.
